박스쌓기와 비슷하다
## 특징 
- 선입후출(First In Last Out) 혹은 후입선출(Last In First Out) 구조
- 데이터를 하나씩만 넣고 뺄 수 있다
- [DFS](탐색.md#DFS) 에서 사용된다

## 구현
```
import java.util.Stack;

Stack<Integer> stk = new Stack<>(); //선언

stk.push(1); //값추가 1
stk.push(1); //값추가 2

stk.pop(); //값제거 및 반환 2

stk.peek(); // 마지막 요소를 반환, 스택 변화 x
stk.clear(); //값 모두 제거
```
