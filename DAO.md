# DAO
Data Access Object
database에 접근하는 역할을 하는 객체
서비스 모델에 해당하는 부분과 데이터베이스를 연결하는 역할
CRUD 작업을 생하는 클래스, 데이터에 대한 CRUD 기능을 전담하는 Object

## 왜 사용하는가
효율적인 커넥션 관리와 보안성
비지니스 로직을 분리


### 개인적인 생각
비지니스 로직을 분리하고 작은 개념으로 쪼개 나눈 이유는 SRP를 위해서도 있지 않은가 라는 생각이 든다
