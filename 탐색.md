# 탐색
많은 양의 데이터 중에서 원하는 데이터를 찾는 과정
## 자료구조
[[스택]] [[큐]] [[그래프]]
## DFS
Depth - First search 깊이 우선 탐색 
그래프의 깊은 부분을 우선적으로 탐색하는 알고리즘
그래프의 가장 먼 부분부터 탐색
스택 이용

### 과정
1. 탐색 시작 노드를 스택에 삽입하고 방문처리
2. 스택의 최상단 노드에 방문하지 않은 인접 노드가 있다면 그 인접 노드를 스택에 넣고 방문처리, 방문하지 않은 인접 노드가 없다면 스택에서 최상단 노드로 꺼냄
3. 2번 과정을 더이상 수행할 수 없을때까지 반복

### 특징
재귀 함수로 이용시 간결하게 구현
가장 가까운 노드부터 탐색
큐 이용
## BFS
 Breadth First Search 너비 우선 탐색
 가까운 노드부터 탐색
### 과정
1. 탐색 시작 노드를 큐에 삽입하고 방문 처리
2. 큐에서 노드를 꺼내 해당 노드의 인접 노드 중 방문하지 않은 노드를 모두 큐에 삽입하고 방문처리
3. 2번의 과정을 더이상 수행 할 수 없을때까지 반복

### 특징
시간 복잡도 :  **O(N)** 
일반적으로 실제 수행시간은 DFS 보다 좋은 편




DFS/BFS나 완전탐색으로 풀 수 있는 마지노선은 500만개 (5\*$10^6$)
