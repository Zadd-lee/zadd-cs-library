# 이진 탐색
## 순차탐색
앞에서부터 데이터를 하나씩 차례대로 확인

## 이진탐색
>[!info] 중간점의 데이터를 반복해 비교

### 특징
- 내부 데이터가 정렬되어 있어야지만 사용 가능한 알고리즘
- 시간 복잡도 : $O(logN)$ 
- 같은 연산을 반복하기 때문에 재귀 함수 이용해 구현

### 구현
``` 
int binarySearch(arr,int target, int start, int end){
int mid = (start+end)/2;

if(arr[mid]==target) { return mid;}
else if(target < arr[mid]){return binarySearch(arr, target,start,mid-1);}
else{ return binarySearch(arr,target,mid+1,end);}
}
```
