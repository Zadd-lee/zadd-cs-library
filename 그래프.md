## 그래프
노드
간선 : 노드끼리 연결됨
# 표현 방식
### 인접 행렬
2차원 배열로 그래프의 연결 관계를 표현하는 식
각각의 노드와 노드의 관계를 2차원 그래프 연결 방식으로 표현
### 인접 리스트
리스트로 그래프의 연결 관계를 표현하는 방식
노드 i 에 대한 리스트를 만들어줌

# 차이점
## 메모리 측면
인접 행렬 : 노드의 개수 증가, 메모리 낭비
인접리스트 : 연결된 정보만 저장-> 메모리 효율적 사용
## 속도
인접 행렬 : 빠르다
인접 리스트 : 연결된 데이터를 모두 조회하기 때문에 속도 느리다
